There are pipes aligned in circular figure, which are numbered. You want to put balls into the pipes. But the floor below has holes in some places. Don't worry, you can rotate the holes. So, please rotate them so that all balls will not fall down.

Input: Two arguments.

  • True/False list meaning the floor state (False means hole).
  • The list of the numbers into which you are going to put the balls.

Output: The list of the units you need to rotate the floor to meet the condition.

Example: 

checkio([True,False,False,True,True,False,True,False,False,False,True],[0,1]) == [1,8]
checkio([True,False,False,True,True,False,True,False,False,False,True],[0,1,2]) == []
